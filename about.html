Estructura de un programa en C++

Estructura de un programa

En la siguiente imagen se observa el código fuente utilizado para imprimir un mensaje en pantalla.


Explicacion de la Estructura

• Comentarios: Todas las líneas que comienzan con los símbolos // indican un comentario. Estas líneas no tienen significado para el compilador, y no se ejecutan. Son ignoradas en el momento de compilación. Sin embargo, son útiles para el programador, quien las utiliza para entender el código y documentarlo.


Programa para imprimir un mensaje en pantalla

// Directivas de preprocesamiento
// Inclusion de librerias para utilizar metodos y funciones predefinidas
#include <iostream>
// Declaracion del espacio de nombres STD
using namespace std;
// Funcion principal del programa
int main () {
// Uso del metodo cout para imprimir un mensaje
cout<<"Bienvenidos a mi primer programa"<<endl ;
// Indica la terminacion exitosa del programa
return 0;
}

• Directivas de procedimientos: #include <iostream> Todas las instrucciones en C++ que comienzan con el símbolo # se utilizan para indicar las directivas de preprocesamiento. En este caso se está incluyendo una librería. Por eso se utiliza la palabra include, seguido del nombre de la librería entre los símbolos < >. Una librería es un archivo con terminación .h el cual contiene métodos que pueden ser utilizados por el programador. Las librerias más utilizadas en C++ son la iostream y cstdio, ambas son útiles para leer y escribir a la consola de ejecución. Recuerda siempre incluir la librería iostream al principio de cada uno de tus programas para poder utilizar las instrucciones cin y cout.

• Uso de espacios de nombres: using namespace std; Esta instrucción se requiere para indicar al compilador que se están usando métodos bajo el espacio de nombres STD, de lo contrario el compilador no entiende los métodos.

Programa para imprimir un mensaje en pantalla

#include <iostream>

using namespace std;
int main () {
cout<<"Hola, estudiantes de nivelacion de la carrera de TICS"<<endl;
cout<<"Bienvenidos al mundo de la programacion en C++"<<endl;
cout<<"Les deseamos mucho exito en su viaje de aprendizaje"<<endl;
return 0;
}

• Funcion principal: Un programa en C++ contiene métodos y funciones, los cuales se utilizan para ejecutar segmentos de código. La función principal se conoce como main().

La estructura es:

int main() {
//Conjunto de instrucciones
return 0;
}

• La sentencia int main() indica la declaración de la función. La palabra int al principio de la función indica que se espera que al terminar la función regrese un valor entero.
