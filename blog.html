Estructuras de control

¿Qué son?

Las estructuras de control se utilizan para controlar el flujo de un programa, son métodos que permiten especificar el orden en el cual se ejecutarán las instrucciones en un algoritmo.

Estructuras

Condicionales

• if

La sentencia if nos permite elegir si se ejecuta o no un bloque de instrucciones de la siguiente manera:

if(condición)
{
sentencias;
sentencias;//si la condición es afirmativa
}
else
{
sentencias;
sentencias;//si la condición es negativa
}

• switch

Selección múltiple con la sentencia switch, permite seleccionar entre varias alternativas posibles así:

switch(expresión)
{
case1: sentencia1;
break;
case2: sentencia2;
break;
-----------
else
caseN: sentenciaN;
break;
default: sentencia;
}
